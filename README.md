GameFAQs Post and Track
======================================
Adds a "Post Message, Track Topic" button to quick post and "Post New Message" forms for untracked topics. Using this button will post your message without preview and automatically start tracking said topic.

Use with one of the following browser extensions:

### Mozilla Firefox ###
*	[Greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)
*	[Scriptish](https://addons.mozilla.org/en-US/firefox/addon/scriptish/)

### Google Chrome ###
*	[Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
*	Convert script to Google Chrome extension:

	>1. Save the script (the `*.user.js` file) to your computer.
	>2. In Chrome, go to _Tools > Extensions_.
	>3. Drag/drop the user script file onto the _Extensions_ page.
